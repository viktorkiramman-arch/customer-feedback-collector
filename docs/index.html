Private by design

Security is part of the workflow, not an afterthought.

FeedbackLoop validates untrusted input on the server, refreshes current authorization state for protected actions, and keeps organization data isolated throughout the application.

  • Current membership and role checks for protected operations
  • Database-backed rate limiting and idempotency
  • Tenant-scoped Prisma access and audit records
  • Formula-safe exports without anti-abuse data
